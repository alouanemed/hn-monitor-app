### Setup

- Create a PostgreSQL role if you don't have one
<br>`create role username login createdb;`
- Install the required gems list
<br>`bundle install`
- Set up the database
<br>`rake db:create`
<br>` rake db:migrate`
- Run the app
<br>`rails s`
